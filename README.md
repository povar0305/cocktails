# Поиск и подбор коктеля

Это приложение помогает находить и просматривать рецепты коктейлей, фильтровать по ингредиентам и категориям, а также получать персональные рекомендации по результатам опроса.

---

## Основные функции
- Поиск коктейлей по названию
- Фильтрация по ингредиентам, категориям и популярности
- Просмотр рецептов, со списком ингредиентов и инструкцией, историей, фактами
- Проведение опросов для получения персональных рекомендаций
- Добавление коктейлей в избранное

## Используемые технологии и библиотеки

В этом проекте применяются следующие инструменты и библиотеки:

- **Nuxt.js** — современный фреймворк на базе Vue.js, который помогает быстро создавать сайты и приложения, умеет рендерить их на сервере и генерировать статические страницы.

- **Vue.js** — основная библиотека для построения интерфейсов, которая делает страницы динамическими и отзывчивыми.

- **Vue Router** — инструмент для управления переходами между страницами и URL в приложении.

- **Pinia** — современная и удобная библиотека для хранения и управления состоянием приложения.

- **Tailwind CSS** — популярный CSS-фреймворк, который позволяет быстро делать красивые и адаптивные интерфейсы с помощью утилитарных классов.

- **PostCSS с Autoprefixer** — автоматический инструмент для добавления нужных префиксов в стили, чтобы они хорошо работали во всех браузерах.

- **Sass** — расширение CSS с возможностями переменных, вложенности и миксинов для удобства стилизации.

- **TypeScript** — язык программирования, основанный на JavaScript, с системой типов, делая код более надежным и понятным.

- **ESLint** — программа для проверки качества кода и соблюдения правил стиля.

- **Husky** — автоматизация проверки кода перед сохранением изменений (например, запуск ESLint).

- **PrimeVue** и **PrimeIcons** — готовый набор интерфейсных компонентов и иконок, чтобы быстро делать красивые кнопки, модальные окна и таблицы.

- **vue-final-modal** — инструмент для создания всплывающих окон и диалогов.

- **@vueuse/core** — коллекция полезных функций и хуков для Vue, упрощающих работу с реактивностью и событиями.

## Открытое API

Данное приложение использует API [Cocktails API](https://apmix.ru/swagger), предоставляющее публичный доступ к базе данных коктейлей. API позволяет:

- Получать список коктейлей по названию, ингредиентам или категориям
- Получать детали конкретных рецептов
- Искать коктейли по различным фильтрам
- Получать список ингредиентов и категорий

## Взаимодействие с проектом
Установка приложения и зависимостей:
```bash
# npm
npm install
```

Запуск приложения для локального развертывания  `http://localhost:3000`:
```bash
# npm
npm run dev
```

Сборка приложения
```bash
# npm
npm run build
```

Предарительная локальная сборка билда
```bash
# npm
npm run preview
```
